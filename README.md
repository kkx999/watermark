# 🛡️ 图片水印 Pro (Watermark Pro)

> 一个安全、快速、纯本地处理的在线图片水印工具。
>
> **无需上传图片到服务器，保护您的隐私安全。**

[👉 **点击这里立即使用**](https://kkx999.github.io/watermark)

---

## ✨ 功能特点

* **🔒 隐私安全**：所有图片处理逻辑均在本地浏览器（Client-side）完成，**原图绝不会上传**至任何服务器。
* **⚡ 极速处理**：基于 HTML5 Canvas 技术，毫秒级生成，无网络延迟。
* **🎨 多种模式**：
    * **全屏平铺**：防盗图必备，自动计算旋转角度与间距，无死角覆盖。
    * **居中显示**：适合 Logo 展示。
    * **右下角署名**：适合摄影作品版权声明。
* **🎛 高度可定制**：支持调节文字内容、大小、颜色、透明度、旋转角度及平铺密度。
* **🌗 暗色模式**：自动跟随系统偏好，支持手动一键切换深色/浅色主题。
* **📱 响应式设计**：完美适配桌面端与移动端，支持拖拽上传。

## 🚀 如何使用

1.  打开 [在线链接](https://kkx999.github.io/watermark)。
2.  点击上传区域或直接将图片拖入网页。
3.  在左侧面板调整水印文字、颜色、角度等参数。
4.  实时预览效果，满意后点击 **"下载处理后的图片"**。

## 🛠️ 技术栈

本项目是一个纯静态网页，无需后端环境。

* **HTML5**
* **CSS3** (CSS Variables, Flexbox)
* **JavaScript** (ES6+, Canvas API)
